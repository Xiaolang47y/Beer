# Beer
Beer (**B**e **E**mulator mor**E** **R**uined) is a Qt6-based GUI frontend that uses Proton to run Windows programs.

This program borrows the Proton translation layer downloaded from Steam to run Windows programs, and provides some convenient tools for launching and management.

The original intention behind creating this program was to solve the difficulties of obtaining and managing Wine across various Linux distributions.

This application was generated using TRAE AI; please use it with caution.

Cheers! 🍺
### Usage
#### Launching the Application
You can start the program in two ways:

The first: download the source code and directly execute `bash beer.sh` to run it.

The second (recommended): download the `Beer` executable from the Release page and double-click it to run.

*Reminder: If you start using the source code method, the configuration files will be saved in the same directory; if you start using the executable, the configuration files will be saved in ~/.config/Beer.*
#### Installing Proton
Install Steam, select the Tools option in the filter on the upper right side of the Library page, search for Proton, and choose a version to install.

After downloading, you will be able to see all the Proton versions you have downloaded on the home page of this application.

If you are using a third-party version of Proton (such as Proton-GE), you can check "Custom Proton directory" in Settings - General Settings within the application, and then select the Proton directory in your file manager. When "Custom Proton directory" is enabled, the Proton version selection on the home page will be disabled.
#### Launching Windows Applications
Set the directory and other options on the home page, then click the Run button below.

Click Save Configuration to save your application configuration in the Saved Applications page; double-click the icon to open it.

Click Generate Desktop File to generate a .desktop file in the desktop folder, allowing you to launch it without using Beer.
